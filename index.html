Welkom op het E-Portfolio van Robin Anthonissen

Student Cloud & Cyber Security aan Thomas More Geel.

Ik heb een enorme passie voor IT-infrastructuur en beveiliging. Mijn weg naar deze passie was uniek: ik ben gestart in een praktische vooropleiding (BSO) en heb vanuit die basis de bewuste en gedreven overstap gemaakt naar een academische IT-bachelor in Cloud & Cyber Security.

Deze transitie toont mijn doorzettingsvermogen en mijn vermogen om theoretische concepten te koppelen aan een praktische mindset. Ik ben constant op zoek naar manieren om complexe systemen veiliger en efficiënter te maken.

Over Mij

Mijn Verhaal

Mijn passie voor IT is diep geworteld in een praktische nieuwsgierigheid. Na mijn BSO vooropleiding besloot ik mezelf uit te dagen door de overstap te maken naar de bachelor Cloud & Cyber Security aan de Thomas More Hogeschool. Deze weg heeft me niet alleen technische kennis bijgebracht, maar ook een enorme dosis doorzettingsvermogen en een hands-on mentaliteit die ik elke dag toepas in mijn projecten.

Projecten & Realisaties

Top Project

Slimme Vuilbak (SKIL2 Semester 1)

Context: Projectweek waarbij we een slimme vuilbak hebben gemaakt die automatisch afval sorteert.

Mijn bijdrage: Ik heb het DevOps-gedeelte verzorgd (zie de map Raspberry Pi/Docker) en de infrastructuur opgezet, terwijl mijn teamgenoten zich richtten op de IoT-hardware en sensoren.

Technologieën: Python, Docker, FastAPI, PostgreSQL, JavaScript, AI, IoT.

Bekijk op GitHub

IT Solution voor Tanzania (SKIL2 Semester 2)

Context: Eén dag de tijd om een IT-solution te bedenken/maken voor mensen zonder internet in Tanzania.

Mijn bijdrage: Ik was verantwoordelijk voor het opzetten van de FastAPI-backend en de database binnen Docker-containers. Om een realistische testomgeving te creëren, draaide dit op een VM die toegankelijk was via VPN voor mijn teamgenoten. Hierdoor konden zij zich volledig focussen op de IoT-implementatie en SMS-functionaliteit.

Technologieën: FastAPI, PostgreSQL, OpenRouter (AI API-keys), Docker.

Bekijk op GitHub

EduStreakz

Mijn bijdrage: Ik heb binnen dit team gewerkt aan zowel de back-end als de front-end. Mijn focus lag op de algemene structuur; de specifieke game-logica heb ik niet gewijzigd.

FullStack FastAPI Site

Context: Project 02 (eerste jaar). Mijn eerste ervaring met het bouwen van een dynamische site met FastAPI.

Mijn bijdrage: In dit project was ik verantwoordelijk voor het ontwikkelen van twee specifieke webpagina's en de bijbehorende FastAPI-backend. Om de samenwerking soepel te laten verlopen, hebben we gewerkt met een duidelijke mappenstructuur per student (bijv. student2_models.py) om zo de code-integriteit te waarborgen.

Technologieën: Python, FastAPI, HTML/CSS, PostgreSQL.
